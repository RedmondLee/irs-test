第39问：信息系统的运行维护都包括哪些内容？
答：包括
    1. 硬件维护：定期检查与更换。
    2. 软件维护：正确性维护、完整性维护、适应性维护。
    3. 数据维护：主要由数据库管理员处理冲突与冗余等。

第40问：假定单位指定你来负责一项信息系统的建设，你将如何做好项目管理工作？
答：
    管理层次上，解决好计划制定、组织团队、项目控制等基本内容。
    项目建设上，解决时间、资源、用户、目标四要素。
    具体落实上，分为项目授权、需求分析、技术选型、实施、评估与分析五步走。

第41问：信息系统项目有哪些特点？
答：（常见互联网项目缺陷和特点）
    1. 目标不精确，任务边界模糊。
    2. 用户需求随项目进度经常改变。
    3. 同时具备智力密集型和劳动密集型特点。
    4. 建设过程中会产生大量文档，对系统维护和升级十分重要。

第42问：什么是信息资源标准化，它有什么作用？
答：
    标准化实质对信息资源的内容和形式做统一的规定，经有官方协商一致，主管机构批准后发布，作为信息资源加工、储存、开发和利用时共同遵守的准则和依据。
    重要性主要体现在：
    1. 标准化是信息系统建设的基本前提
    2. 能保障系统的运行效率
    3. 标准化后更易实现资源共享。

第43问：概述信息资源标准的内容。
答：
    主要包含以下几个部分：
    1. 内容标准：
        a) 信息编码标准：UNICODE,UTF-8/16,GBK13000等
        b) 数据格式：包含各种文本格式、图像格式、音频格式、视频格式（以及其子格式）
        c) 信息标志：一种永久性资源符号，唯一标志数据，与物理位置无关。所有系统都应该构建标志系统。
    2. 描述标准：
        a) 元数据：管理型元数据（合法性）、描述型元数据（索引）、技术型元数据等
        b) 主要元数据规范有DC/NISO(美国国家信息标准局)/IEEE LOM（即所谓学习对象）等规范。

第44问：简述企业资源管理出现的背景、发展过程及主要阶段的特点。
答：
    背景为社会经济结构变化，从从业经济向信息经济过度。工作性质和经营方式变化，从物质劳动转向智力劳动。企业经营环境和机制变化，从地理限制转向信息交换，越来越多采用信息技术和现代化设备。
    发展阶段分为五个。
    1. 文书管理阶段（paperwork management stage）：重点是对物理载体（如书籍）的管理
    2. 自动化技术管理阶段：20世纪60-70年代，初步使用文字处理中心等电子技术。
    3. 信息资源管理阶段：20世纪80年代中期，重点是注重信息系统与管理理论的结合，如质量控制、计算机系统审计等。
    4. 商业竞争和分析阶段：20世纪90年代，新技术层出不穷，重视竞争分析和信息利用。
    5. 战略管理阶段：21世纪后，IT依赖性提高，和IT对企业各层次作业和管理决策的渗透加剧（accelerate）。


第45问：阐述企业信息资源管理的主要内容、目标、任务和工作模式。
答：
    内涵：
        广义内涵包括所有具有与信息相关的知识的人才、硬件和软件设备、信息系统的服务和结构。
        管理企业信息源：包括外部信息源（政策、法律法规、经济信息等）和内部信息源（市场监测信息、财务管理信息等、技术研发信息等）
    目标：
        综合目标追求3E：效益（Efficient），效率（Effective）和经济（Ecnomical）。具体包括：
        1. 为企业构建信息化标准平台。
        2. 确保高效管理流程。
        3. 确保企业信息安全。
        4. 确保有效利用和高效分析信息等。
    任务：
        1. 制定企业资源规划
        2. 组件信息资源管理架构
        3. 组织技术维护、设备升级、人员培训。
        4. 建立管理制度。
        5. 综合利用信息辅助决策。
    工作模式：
        1. 集中式与分散式。（即企业统一构建信息平台，或各部门组织各自构建信息平台）
        2. 流程模式有四步式，五步式，六步模式（确定需求、分析需求、确定信息库、组件管理系统、制定决策、开发）等。


第46问：何谓企业信息资源规划？它包括哪些主要内容？
答：
    一个有效的信息资源管理规划可以提高生产效率、降低成本、改善管理决策。
    主要包含的内容有：
    1. 需求分析。
    2. 系统架构设计。
    3. 专用软件和数据概念模型。
    4. 信息安全评估。
    5. 实现分布式处理等。


第47问：为什么要进行信息审计（audit）？它的作用和主要方法是什么？
答：
    信息审计是为了解决IT管理者与业务管理者之间的协调问题，解决不同信息系统之间缺乏整合的问题，避免出现数据冗余和工作重复。
    审计的方法由五个阶段组成：
    1. 宣传启动（promote）阶段：促进组织合作，协调审计人员，形成初步框架
    2. 调查（Identify）阶段：定义组织任务、识别组织环境、识别组织结构、信息流、信息资源。
    3. 分析（Analyse）阶段：评估信息资源内容，绘制详细流程图，编写初步报告。
    4. 会计核算（account）阶段：对信息内容、价值、成本进行核算。
    5. 综合（synthesise）阶段：综合审计报告并制定战略策略。
    

第48问：何谓环境扫描？它有哪些层次和主要活动？
答：
    环境扫描是对组织外部环境发生的事件、趋势和关系进行信息采集和分析，帮助组织规划未来行动。
    它可以分为高、低两个层次。
    高层次主要监测整个环境，识别出需要密切关注的领域。低层次负责监测特定领域，并仔细加以分析。
    它的主要活动包括：竞争情报（competitive intelligence）商业智能（bussiness intelligence）问题管理（Issue management），社会智能（social intelligence）


第49问：企业信息化的含义和基本任务是什么？
答：
    一般来说其内容包括以下几方面：
    1. 目的是提高企业经济效益和竞争力。
    2. 技术手段是构建软硬件平台
    3. 非技术手段是业务流程重组、人员培训等。
    基本任务包括以下几个方面：
    1. 生产过程信息化：用信息化、智能化手段解决生产加工中的复杂问题
    2. 研究开发工作信息化：计算机辅助设计。
    3. 其他信息化，包括产品信息化、涉及信息化、市场营销信息化、财务和管理信息化等。

第50问：试评述企业信息化的发展历程？
答：
    内容和概念较多较冷门，参考第44问。

第51问：简述企业信息化规划的主要内容，分析一个企业信息系统建设的策略。
答：
    企业信息规划主要是从企业自身战略目标出发，综合分析企业生产能力、技术水平、管理方法、人员要素和市场竞争力等多方面，逐个分析目标需要什么信息技术的过程。
    通常包含若干子规划：
    1. 统一的信息化平台规划。
    2. 应用系统规划。
    3. 综合应用规划：协调生产和分析平台。
    实施策略包括：
    1. 组织管理
    2. 系统选型
    3. 业务流程重组
    4. 涉及与实施
    5. 运行与维护
    这几个步骤和方面。

第52问：业务流程重组有何必要性和作用？简述BPR的方法和步骤。
答：
    业务流程重组（BPR）是指基于现有优势、需求和人员、技术等要素，对业务流程整体重新思考并重新架构，以期降低成本，提高效率的过程。
    包括几个步骤：
    1. 业务流程的结构分析。
    2. 业务流程的内涵分析：了解每个流程的事件、状态。
    3. 流程再造：包含清除、简化、自动化。

第53问：试分析评述企业信息化的发展趋势。
答：
    目前技术大潮在对传统商业模式产生新的挑战。
    趋势包括
    1. 移动化
    2. 社交化
    3. 云计算和云制造。
    4. 大数据和数据驱动。
    5. 人工智能协助。
    总结：技术的参与比重越来越高，信息安全和合规的重要性增大，战略决策与企业经营、生产过程深度融合。

第54问：简述政府信息资源管理产生的时代背景和技术背景。
答：
    早期是文书管理时代，我国可以追溯到周朝时期。但随着进入20世纪70-80年代，社会环境和科学技术产生重大变化，政府运作过程中海量的纸质文书问题引起西方国家关注，信息资源作为一种独立资源的理念得到普遍认同。同时，随着技术进步，进一步引起了信息资源管理革命。
    是二者互相融合的结果。
    时代方面主要是
    1. 信息资源观念的转变
    2. 经济信息化的推进
    3. 政府体制改革的需要
    技术方面：
    1. 计算机硬件蓬勃发展并逐渐引入企业和政府管理
    2. 以数据库为代表的软件技术快速发展
    3. 互联网技术不断深化    

第55问：简述政府信息资源管理演进的主要阶段
答：
    1. 政府记录管理时期：早期通过不断立法和机构组织改革完善记录制度，但随着进入20世纪后文书逐渐增多，工作也越来越复杂。 电子化时期： 20世纪70年代，随着国会报告显示每年联邦政府产生的庞大记录给工作带来极大不便，开始对其进行电子化改革。
    2. 政府信息技术管理时期： 20世纪80年代以后，计算机和通信技术被逐渐引入征服管理。自动化技术极大改进了政府工作效率，一系列政府数据管理项目和技术得以陆续开发和实施（大型数据库等）
    3. 政府信息资源管理时期：随着技术不断革新，但落后的管理体制阻碍政府机构的运行。并且越来越多学者认识到以信息技术和信息内容为主的狭义信息资源管理局限性越来越大，失败案例越来越多，政府进入资源整合管理时期。
    4. 电子政府：2000年以后，为迎接新世纪政府职能现代化、民主化、公开化和效率化的需求，政府进入电子政务时代，更强调利用先进网络与通信技术将政府功能和信息集成。


第56问：政府信息资源具有哪些基本特征？
答：
    基本特点：
    1. 范围广
    2. 数量大
    3. 价值高
    4. 储存分散
    利用特点：
    1. 受众普适性
    2. 资源供给的义务性
    3. 管理主体的自利性与管理客体的公共性
    经济属性：
    1. 显著的正外部性（positive externality）（投资使社会整体获益）
    2. 公共物品属性：非竞争、非排他、公共。

第57问：政府信息资源管理的主要内容和任务有哪些？
答：
    内容是指与政府信息资源开发和利用有关的决策、计划、预算、组织、指导和培训等活动，具体包括
    1. 政府信息化政策的制定。
    2. 政府信息化项目的管理
    3. 政府信息基础设施的建设
    4. 政府信息技术的管理。
    5. 信息化人力资源开发
    6. 信息法律法规建设

    任务包括：
    1. 对政府信息进行综合管理
    2. 保证信息资源真实、准确、科学。
    3. 树立政府公众形象。


第58问：简述政府信息化与电子政务建设的关系。
答：
    政府信息化是指用信息技术全面推进政府部门办公自动化、网络化，以及信息资源共享、利用以提高政府决策水平，籍贯工作效率的进程。
    电子政务（electronic goverment）则是在政府内部行政电子化的基础上，利用计算机和网络技术建立政府信息系统，并为企业、社会组织和公众提供服务。

第59问：我国政府信息化建设面临的困境和挑战。
答：

    随着信息化加深，社会变化，我国政府信息化建设具有信息大而全，共享需求高，发展需求大的特点。
    由此带来的挑战主要分以下几点：
    1. 信息资源管理法规体系有待进一步完善
    2. 数据多元互动体系有待强化，与社会机构沟通待健全。
    3. 急需加强数据安全方面的新技术应用
